# Nagare Project Introduction Website

This is a standalone, frontend-only website for the Nagare project. It provides an overview of the system, its features, and installation instructions.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Build & Deploy

```bash
# Build the production-ready site
npm run build
```

The output will be in the `dist/` directory. You can host this directory on any static web server, such as GitHub Pages, Netlify, or Vercel.

### Deploying to GitHub Pages

1. Build the project: `npm run build`
2. Push the contents of the `dist` folder to a `gh-pages` branch.
3. Enable GitHub Pages in your repository settings.
